gallery-sm-tree Module

Defined in: src/sm-tree/js/tree.js:7

Provides the Tree class.

This module provides the following classes:

  • Plugin.Tree.Lazy
  • Tree
  • Tree.Node

This module is a rollup of the following modules:

  • gallery-sm-tree-lazy
    Provides Plugin.Tree.Lazy, a plugin for Tree that makes it easy to lazily load and populate the contents of tree nodes the first time they're opened.
  • gallery-sm-tree-node
    Provides the Tree.Node class, which represents a tree node contained in a Tree data structure.
